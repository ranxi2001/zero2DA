# zero2DA

从零入门数据分析 / 商业分析 / 商业智能 —— 面向转行者与初学者的系统化教程。

## 学习路线

1. **DA 基础** — 数据分析师的定义、核心思维、岗位对比
2. **数据思维** — 假设驱动、指标体系、漏斗分析、A/B 测试
3. **SQL** — 从基础语法到窗口函数，覆盖面试高频题型
4. **Excel** — 数据清洗、透视表、公式、图表
5. **Python** — Pandas、Matplotlib、数据分析实战
6. **可视化 & BI** — Tableau、Power BI、数据看板设计
7. **商业分析** — 商业模式、用户行为、营收分析
8. **DA 面试** — SQL 面试题、Case Study、真题解析
9. **工具箱** — 数据分析工具全景图与学习资源
10. **实战项目** — 能放进简历的完整分析项目

## 本地开发

```bash
gem install bundler jekyll
bundle install
bundle exec jekyll serve
# 访问 http://localhost:4000/zero2DA
```

## 技术栈

- Jekyll 静态站点
- GitHub Pages 部署
- 纯 CSS + Vanilla JS
