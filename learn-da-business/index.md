# 商业分析

数据分析的终极价值在于**驱动商业决策**。技术能力决定了你能不能取到数据、处理数据，而商业分析能力决定了你能不能从数据中看到"故事"，并把它翻译成可执行的业务动作。

本模块将带你从**商业模式**出发，理解一个公司到底怎么赚钱；再深入**用户行为**，用数据刻画用户从哪里来、在产品里做了什么、为什么流失；然后进入**营收分析**，学习如何拆解收入结构并做出合理预测；最后通过**竞品分析**，掌握在数据层面衡量市场格局的方法。

## 学习建议

> **前置知识**：建议先完成"数据思维"与"SQL"模块。商业分析依赖你对指标体系、漏斗分析、A/B 测试等概念的理解，同时需要用 SQL 从数据库中提取业务数据。

- 每篇文章都附带真实场景的分析案例，建议边读边思考"如果我是这家公司的分析师，我会怎么拆解这个问题"
- 商业分析没有标准答案，核心是**逻辑自洽 + 数据支撑**
- 尝试用文章中的框架分析你日常使用的产品（比如美团、抖音、Netflix），这是最好的练习方式

## 这个模块对求职的价值

在数据分析师面试中，商业分析能力往往是区分"会写 SQL 的人"和"能独立做分析的人"的关键。面试官常问的 Case Study、产品指标设计、开放性业务问题，本质上都是在考察你的商业分析功底。

学完本模块后，你将能够：

- 快速理解一个新业务的商业逻辑和核心指标
- 用数据驱动的方式分析用户行为并提出优化建议
- 拆解收入结构、识别增长瓶颈、做出合理的营收预测
- 从数据角度评估竞争格局并支持战略决策
