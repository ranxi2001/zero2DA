# DA 基础：建立正确的数据分析师认知

> 在学习 SQL、Python、BI 工具之前，先搞清楚一个问题——**数据分析师到底是做什么的？**

很多人一上来就啃工具，学了半年发现方向不对、岗位选错、面试答不上"你理解的数据分析是什么"。这个模块用 5 篇文章帮你建立对数据分析师岗位的系统认知，是后续所有模块的基础。

## 学完你会获得

- 能清晰地向别人（或面试官）解释"什么是数据分析师"
- 理解数据分析的核心思维方式，而不仅仅是工具操作
- 区分 BA、DA、BI、Data Scientist、Data Engineer 等角色
- 了解数据分析师的真实日常，知道自己未来的工作状态
- 根据自身背景选择最适合的行业和方向

## 适合谁

- **零基础转行者**：想做数据分析但还不了解这个岗位
- **在校学生**：正在考虑数据分析方向，想提前建立认知
- **已经在学工具的人**：学了 SQL / Python 但不知道怎么用到工作中
