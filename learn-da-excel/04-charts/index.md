# 图表制作与美化

> 数据分析师的最终产出往往是一张图。一张好图能在 3 秒内传递关键信息，一张差图只会让人更困惑。

本篇将带你从"会做图"到"做好图"——不仅掌握 Excel 图表的制作方法，更要学会如何让图表**清晰、准确、专业**。

---

## 图表选择原则：先想清楚要表达什么

在动手做图之前，先回答一个问题：**这张图要传递什么信息？**

| 要表达的关系 | 推荐图表 | 典型场景 |
|-------------|---------|---------|
| 趋势变化 | 折线图 | 月度 GMV 趋势、DAU 变化 |
| 大小比较 | 柱状图/条形图 | 各城市销售额排名 |
| 占比构成 | 饼图/环形图/堆叠柱状图 | 各渠道流量占比 |
| 分布情况 | 直方图/箱线图 | 用户年龄分布、订单金额分布 |
| 相关关系 | 散点图 | 广告投放 vs 转化率的关系 |
| 多指标对比 | 组合图（柱状+折线） | 销售额（柱状）+ 增长率（折线） |

> **黄金法则**：如果你不确定用什么图，**柱状图和折线图能解决 80% 的需求**。

## 柱状图——最常用的比较图表

### 基础柱状图

**适用场景**：比较不同类别的数值大小。

**案例：各城市 Q1 销售额对比**

| 城市 | 销售额（万元） |
|------|-------------|
| 北京 | 1,258 |
| 上海 | 1,105 |
| 广州 | 876 |
| 深圳 | 923 |
| 成都 | 456 |

**操作步骤**：

1. 选中数据区域（含表头）
2. `插入` → `图表` → `簇状柱形图`
3. 柱状图自动生成

**美化技巧**：

| 操作 | 目的 |
|------|------|
| 按数值降序排列 | 让最大值在最左边，便于比较 |
| 删除网格线 | 减少视觉噪声 |
| 添加数据标签 | 直接在柱子上方显示数值，无需读者自己对坐标轴 |
| 统一颜色 | 用同一颜色，最大值用强调色高亮 |
| 删除图例 | 只有一个系列时不需要图例 |
| 调整间距 | 柱子间距设为 80%-120%，柱子不要太挤也不要太散 |

### 分组柱状图

**适用场景**：比较不同类别在不同分组下的表现。

**案例：各城市线上 vs 线下销售额对比**

| 城市 | 线上 | 线下 |
|------|------|------|
| 北京 | 480 | 778 |
| 上海 | 520 | 585 |
| 广州 | 350 | 526 |

**操作**：选中含两个系列的数据 → 插入簇状柱形图。

**美化技巧**：

- 线上用一种颜色（如蓝色），线下用另一种颜色（如灰色）
- 如果分析重点是线上，让线上的颜色更鲜艳
- 图例放在图表上方或右侧

### 堆叠柱状图

**适用场景**：既看总量又看各部分占比。

**案例：各月销售额的品类构成**

| 月份 | 电子产品 | 服装 | 食品 | 家居 |
|------|---------|------|------|------|
| 1月 | 1,200 | 900 | 600 | 500 |
| 2月 | 1,000 | 850 | 550 | 400 |
| 3月 | 1,400 | 1,000 | 700 | 500 |

**操作**：选中数据 → 插入堆叠柱形图。

**注意**：堆叠柱状图适合看总量变化和大致占比，但**不适合精确比较中间部分**（因为底部不对齐）。如果需要精确对比每个品类，用分组柱状图更好。

## 折线图——展示趋势的首选

### 基础折线图

**适用场景**：展示指标随时间的变化趋势。

**案例：月度 DAU 趋势**

| 月份 | DAU（万） |
|------|---------|
| 2024-10 | 150 |
| 2024-11 | 162 |
| 2024-12 | 175 |
| 2025-01 | 168 |
| 2025-02 | 155 |
| 2025-03 | 182 |

**美化技巧**：

| 操作 | 目的 |
|------|------|
| 添加数据标记（圆点） | 让每个数据点清晰可见 |
| Y 轴不从 0 开始 | 如果数据变化幅度小，可以截断 Y 轴以放大趋势（但要标注） |
| 添加趋势线 | 右键数据系列 → 添加趋势线 → 选择线性/移动平均 |
| 标注关键事件 | 用文本框标注"大促"、"版本更新"等关键事件 |
| 线条粗细 | 设为 2-3pt，太细看不清，太粗显得粗糙 |

### 多折线对比

**适用场景**：对比多个指标或多个分组的趋势。

**案例：三条产品线的月度销售趋势**

**美化技巧**：

- **最多 4 条线**：超过 4 条线图表会很混乱
- **颜色要有区分度**：不要用同一色系的深浅色
- **标注直接放在线尾**：比图例更直观，读者不用来回对照
- **突出重点**：要强调的线用粗线+鲜艳色，其他用细线+灰色

## 组合图——柱状图 + 折线图的黄金搭档

### 适用场景

当你需要同时展示**量级差异很大的两个指标**时，用组合图：

- 销售额（柱状图，左 Y 轴）+ 增长率（折线图，右 Y 轴）
- 订单量（柱状图）+ 转化率（折线图）
- 用户数（柱状图）+ ARPU（折线图）

### 案例：月度销售额与环比增长率

| 月份 | 销售额（万元） | 环比增长率 |
|------|-------------|-----------|
| 1月 | 3,200 | — |
| 2月 | 2,800 | -12.5% |
| 3月 | 3,600 | +28.6% |
| 4月 | 3,800 | +5.6% |
| 5月 | 4,200 | +10.5% |
| 6月 | 4,000 | -4.8% |

**操作步骤**：

1. 选中数据 → 插入簇状柱形图
2. 右键"增长率"系列 → `更改系列图表类型` → 选择"折线图"
3. 勾选"次坐标轴"
4. 调整右 Y 轴的范围

**美化技巧**：

- 柱状图用填充色（如蓝色），折线图用对比色（如橙色）
- 左 Y 轴标注"销售额（万元）"，右 Y 轴标注"环比增长率"
- 增长率的折线图上添加数据标签，方便直接阅读

## 饼图——谨慎使用

### 适用场景

展示各部分占整体的比例。

### 制作要点

| 要点 | 说明 |
|------|------|
| **类别不超过 5 个** | 超过 5 个就把小的合并为"其他" |
| **从 12 点钟方向开始** | 最大的部分从顶部开始，顺时针排列 |
| **添加数据标签** | 显示百分比和类别名称 |
| **不要用 3D 饼图** | 3D 效果会扭曲比例，误导读者 |
| **考虑用环形图替代** | 中间可以放总计数字 |

### 什么时候不要用饼图

- 类别超过 5 个 → 用柱状图
- 需要对比两个饼图 → 用分组柱状图
- 各类别比例相近 → 用柱状图（饼图难以区分相近的角度）

> **数据可视化专家的建议**：大多数情况下，柱状图比饼图更好。饼图唯一的优势是直观展示"部分与整体的关系"。

## 专业美化：让图表从"能看"到"好看"

### 配色原则

**原则一：少即是多**

- 单一系列用同一颜色（可以高亮重点值）
- 多系列控制在 3-5 种颜色以内
- 避免使用 Excel 默认的鲜艳配色

**推荐配色方案**：

| 风格 | 颜色组合 | 适用场景 |
|------|---------|---------|
| 商务蓝 | #2563EB + #93C5FD + #DBEAFE | 正式报告 |
| 科技感 | #10B981 + #6366F1 + #F59E0B | 互联网公司 |
| 简约灰 | #374151 + #9CA3AF + #E5E7EB | 配合强调色使用 |
| 暖色调 | #EF4444 + #F97316 + #F59E0B | 突出警告/下降趋势 |

**原则二：用颜色讲故事**

- 正面趋势用绿色/蓝色
- 负面趋势用红色/橙色
- 不需要强调的数据用灰色
- 需要强调的数据用饱和色

### 去除多余元素

**要删除的元素**：

| 元素 | 是否保留 |
|------|---------|
| 图表标题 | 保留（但简洁有力） |
| 网格线 | 通常删除，或用极浅的灰色 |
| 图表边框 | 删除 |
| 背景色 | 白色或透明 |
| 3D 效果 | 永远不用 |
| 数据表 | 通常删除（用数据标签替代） |
| 坐标轴标题 | 只在单位不明确时保留 |

### 标题写法

**不好的标题**：

> "2025年Q1各城市销售额柱状图"

**好的标题**：

> "北京 Q1 销售额领跑全国，是成都的 2.8 倍"

好的标题不是描述图表类型，而是**直接说出结论**。

### 数据标签的使用

| 场景 | 建议 |
|------|------|
| 柱状图 | 只标注最大值和最小值，或者全部标注 |
| 折线图 | 标注关键转折点 |
| 饼图 | 必须标注（没有坐标轴参考） |
| 数据点少（<8 个） | 全部标注 |
| 数据点多（>15 个） | 只标注关键点，避免拥挤 |

## 实战案例：制作月度经营分析图表

### 场景

你需要在月度经营分析会上展示以下内容：

1. 本月 GMV 达成情况（vs 目标）
2. 近 6 个月 GMV 趋势
3. 各品类 GMV 占比
4. 各渠道 ROI 对比

### 图表一：GMV 达成进度（子弹图/进度条）

用柱状图模拟进度条：

- 灰色条：目标值（5,000 万）
- 蓝色条：实际值（4,200 万）
- 标注达成率：84%

**操作**：创建一个分组柱状图，让目标值的柱子宽度更大（通过调整系列重叠和间距）。

### 图表二：近 6 个月 GMV 趋势（组合图）

- 柱状图：每月 GMV
- 折线图：环比增长率（右 Y 轴）
- 虚线：目标线

**关键美化**：
- 标注目标线（水平虚线，标注"月目标 5000 万"）
- 未达标的月份用红色柱子
- 达标的月份用蓝色柱子

### 图表三：品类 GMV 占比（环形图）

- 4 个品类，最大的品类在顶部
- 中心显示"总 GMV：4,200 万"
- 每个扇区标注品类名和百分比

### 图表四：各渠道 ROI 对比（条形图）

- 水平条形图，按 ROI 从高到低排列
- 标注 ROI 基准线（如 ROI = 1.0）
- ROI > 1 的用绿色，ROI < 1 的用红色

## 图表制作检查清单

每次做完图表后，用这个清单检查一遍：

| 检查项 | 是否通过 |
|--------|---------|
| 图表类型是否匹配要表达的信息？ |  |
| 标题是否传递了关键结论？ |  |
| 数据是否准确（检查数据源）？ |  |
| 是否删除了多余的元素（网格线、边框等）？ |  |
| 配色是否协调，是否突出了重点？ |  |
| 坐标轴刻度是否合理？Y 轴是否需要从 0 开始？ |  |
| 字体大小是否够大（投影仪上能看清）？ |  |
| 图例位置是否合理？是否可以用直接标注替代？ |  |
| 如果打印成黑白，是否仍然可以区分各系列？ |  |
| 图表是否有误导性（截断的坐标轴、不等距的刻度）？ |  |

## 常见错误与纠正

### 错误一：Y 轴截断但不标注

如果 Y 轴不从 0 开始，会放大趋势变化，可能误导读者。

**纠正**：要么从 0 开始，要么在 Y 轴上明确标注"轴已截断"。

### 错误二：使用 3D 图表

3D 效果会扭曲数据的视觉比例，让前面的数据看起来比实际更大。

**纠正**：永远使用 2D 图表。

### 错误三：颜色太多

6 种不同的颜色让图表像彩虹一样，分散注意力。

**纠正**：主色 1 种 + 强调色 1 种 + 灰色作为背景/对比色。

### 错误四：信息过载

一张图表试图展示 5 个维度的信息。

**纠正**：一图一结论。如果有多个维度，拆成多张图。

### 错误五：饼图类别太多

10 个类别的饼图，最小的几个扇区根本看不清。

**纠正**：合并小类别为"其他"，或者改用柱状图。

## 小结

| 要点 | 说明 |
|------|------|
| 先想后做 | 先确定要表达什么关系，再选图表类型 |
| 常用图表 | 柱状图（比较）、折线图（趋势）、组合图（双指标）、饼图（占比） |
| 美化原则 | 少即是多、用颜色讲故事、标题说结论 |
| 常见错误 | 3D 效果、颜色过多、信息过载、Y 轴截断不标注 |
| 检查清单 | 每次做完图用检查清单过一遍 |
