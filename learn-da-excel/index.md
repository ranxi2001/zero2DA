## 为什么数据分析师还需要学 Excel

你可能会问："都 2025 年了，还学 Excel？"答案是：**必须学，而且要学好**。

### 工作中 Excel 的实际使用场景

| 场景 | 说明 |
|------|------|
| **快速探索数据** | 拿到一个新的 CSV/XLSX 文件，先用 Excel 看看长什么样 |
| **和非技术同事协作** | 运营、财务、市场同事发来的数据几乎都是 Excel |
| **做周报/月报** | 很多公司的日报、周报仍然是 Excel 模板 |
| **数据验证** | 用 SQL 跑出来的结果，用 Excel 交叉验证 |
| **制作简单图表** | 开会前 5 分钟快速出一张图 |
| **面试中的实操环节** | 部分公司的面试会让你现场用 Excel 做分析 |

### 学习建议

- **边学边练**：每篇文章都有实操案例，建议打开 Excel 跟着做
- **记住快捷键**：效率差 10 倍往往就差在快捷键上
- **和 SQL/Python 互补**：大数据集用 SQL，中等数据集用 Python，小数据集和汇报用 Excel
