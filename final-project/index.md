# Final Project

本模块正在紧锣密鼓地筹备中，将带你完成一个**从零到一的完整数据分析项目**——可以直接放进简历和 Portfolio 的实战作品。

## 规划内容

| 阶段 | 内容概要 |
|------|---------|
| 项目选题与数据获取 | 如何选择一个有分析价值的题目、获取真实数据集、定义分析目标 |
| 数据清洗与探索 | 用 SQL 和 Python 进行数据清洗、EDA（探索性数据分析）、数据质量检查 |
| 深入分析 | 用户分群、漏斗分析、留存分析、A/B 测试——运用前面模块学到的方法论 |
| 可视化与 Dashboard | 用 Tableau 或 Power BI 构建交互式数据看板 |
| 分析报告撰写 | 撰写一份专业的分析报告，包含背景、方法、发现和建议 |
| Portfolio 展示 | 将项目整理为 GitHub 仓库和个人网站上的作品展示 |

## 项目目标

完成本模块后，你将拥有：

- 一个完整的数据分析项目代码仓库（SQL + Python + 可视化）
- 一份专业的分析报告
- 一个交互式 Dashboard
- 一段可以在面试中讲 10-15 分钟的项目经历

> 内容正在准备中，敬请期待。建议先完成前面的学习模块，尤其是 [SQL](../learn-da-sql/)、[Python](../learn-da-python/)、[可视化 & BI](../learn-da-bi/) 和 [商业分析](../learn-da-business/) 模块，为实战项目打好基础。
