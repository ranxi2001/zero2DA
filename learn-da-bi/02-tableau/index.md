# Tableau 入门

> **Tableau 是数据分析领域最受欢迎的可视化工具之一。** 它最大的优势是"拖拽式"操作——不需要写代码，把字段拖到画布上就能生成图表。从硅谷科技公司到国内互联网大厂，Tableau 是 DA/BA 岗位 JD 上出现频率最高的 BI 工具之一。

## Tableau 产品家族

| 产品 | 用途 | 价格 |
|------|------|------|
| **Tableau Public** | 免费版，作品必须公开发布 | 免费 |
| **Tableau Desktop** | 专业版，支持私有数据和本地保存 | 付费 |
| **Tableau Server/Cloud** | 企业级部署，团队协作和权限管理 | 企业付费 |
| **Tableau Prep** | 数据清洗和准备工具 | 付费（含在 Creator 许可中） |

> **学习建议**：先用 Tableau Public 学习和练习，功能上和 Desktop 几乎一样，唯一的限制是保存时必须发布到 Tableau Public 网站。求职时在简历上附上你的 Tableau Public 主页链接是一大加分项。

## 安装与界面

### 下载安装

1. 访问 [Tableau Public](https://public.tableau.com/) 注册账号并下载
2. 安装过程一路默认即可
3. 首次启动后，会看到"连接到数据"的起始页面

### 界面核心区域

Tableau 的工作界面分为以下几个关键区域：

| 区域 | 说明 |
|------|------|
| **数据面板**（左侧） | 显示已连接数据源的所有字段，分为"维度"（蓝色）和"度量"（绿色） |
| **画布/视图**（中央） | 图表的绘制区域 |
| **列 / 行 货架** | 将字段拖到"列"或"行"上，定义图表的 X 轴和 Y 轴 |
| **标记卡** | 控制颜色、大小、标签、详细信息、工具提示 |
| **筛选器货架** | 添加筛选条件 |
| **智能推荐** | Tableau 根据你拖入的字段自动推荐图表类型 |

## 连接数据

### 支持的数据源

Tableau 支持几十种数据源，数据分析师最常用的包括：

- **文件**：CSV、Excel、JSON、PDF 表格
- **数据库**：MySQL、PostgreSQL、SQL Server、BigQuery、Snowflake
- **云服务**：Google Sheets、Salesforce、Amazon Redshift

### 连接 CSV/Excel 示例

1. 打开 Tableau → 左侧选择"文本文件"（CSV）或"Microsoft Excel"
2. 选择文件后进入数据预览界面
3. 检查字段类型是否正确（日期、字符串、数字）
4. 如果有多个表，可以通过拖拽建立关联（Join）
5. 点击"工作表 1"进入可视化编辑界面

### 数据类型识别

Tableau 会自动识别字段类型，但有时需要手动调整：

| 图标 | 类型 | 说明 |
|:---:|------|------|
| Abc | 字符串 | 文本类数据 |
| # | 数值 | 整数或小数 |
| 📅 | 日期 | 日期或日期时间 |
| 🌍 | 地理 | 国家、省份、城市 |
| T\|F | 布尔 | True/False |

> **常见问题**：用户 ID、订单编号虽然是数字，但应该设为"字符串"类型。在字段上右键 → "更改数据类型"即可修改。

## 维度 vs 度量

理解"维度"和"度量"是使用 Tableau 的关键：

| 概念 | 定义 | 举例 | 颜色 |
|------|------|------|:---:|
| **维度** | 分类/描述性字段，用来分组 | 渠道、城市、品类、日期 | 蓝色 |
| **度量** | 可以计算的数值字段 | GMV、订单量、UV、转化率 | 绿色 |

简单记忆：**维度决定你"按什么分组"，度量决定你"看什么数字"。**

这和 SQL 的 `GROUP BY` 逻辑完全一致：

```
SQL:    SELECT channel, SUM(gmv) FROM orders GROUP BY channel
Tableau: 把"渠道"拖到行，把"GMV"拖到列
```

## 制作基础图表

### 柱状图：渠道 GMV 对比

1. 将「渠道」拖到**行**
2. 将「GMV」拖到**列**（Tableau 默认对度量做 SUM 聚合）
3. Tableau 自动生成水平条形图
4. 点击工具栏的**降序排序**按钮
5. 将「GMV」再拖到**标记卡 → 标签**，显示具体数值

> 恭喜，你的第一张 Tableau 图表诞生了！

### 折线图：GMV 趋势

1. 将「订单日期」拖到**列**（Tableau 默认按年聚合）
2. 点击日期字段旁的 "+" 号展开到月/日
3. 将「GMV」拖到**行**
4. 图表自动变为折线图
5. 将「渠道」拖到**标记卡 → 颜色**，按渠道分色

### 散点图：UV 与转化率的关系

1. 将「UV」拖到**列**
2. 将「转化率」拖到**行**
3. 将「渠道」拖到**标记卡 → 颜色**
4. 将「GMV」拖到**标记卡 → 大小**（变成气泡图）
5. 将「渠道」拖到**标记卡 → 标签**

### 地图：各城市销售分布

1. 将「城市」拖到**标记卡 → 详细信息**（Tableau 自动识别地理字段并生成地图）
2. 将「GMV」拖到**标记卡 → 大小**
3. 将「GMV」再拖到**标记卡 → 颜色**（按 GMV 大小着色）
4. 调整颜色方案：点击颜色图例 → 编辑颜色 → 选择顺序色板

## 计算字段

计算字段是 Tableau 的核心功能之一，让你在不修改源数据的情况下创建新的指标。

### 创建计算字段

右键数据面板空白处 → "创建计算字段"

### 常用计算示例

```
// 转化率
[订单数] / [UV]

// 客单价
SUM([GMV]) / SUM([订单数])

// 环比增长率
(SUM([GMV]) - LOOKUP(SUM([GMV]), -1)) / LOOKUP(SUM([GMV]), -1)

// 用户分层
IF [总消费] >= 5000 THEN "高价值"
ELSEIF [总消费] >= 1000 THEN "中等价值"
ELSE "低价值"
END

// 日期相关
DATEDIFF('day', [注册日期], [最后活跃日期])

// 字符串处理
IF CONTAINS([渠道], "微信") THEN "微信生态"
ELSEIF CONTAINS([渠道], "抖音") THEN "抖音生态"
ELSE "其他"
END
```

### 表计算

表计算是在 Tableau 视图层面做的计算，不改变底层数据。常见的表计算：

| 表计算 | 用途 |
|-------|------|
| **累计总计** | 月度 GMV 累计趋势 |
| **百分比差异** | 环比/同比增长率 |
| **排名** | 各渠道/品类排名 |
| **移动平均** | 平滑短期波动，看趋势 |
| **占总额百分比** | 各品类 GMV 占比 |

操作方式：右键度量字段 → "快速表计算" → 选择计算类型。

## 筛选器与参数

### 筛选器

将字段拖到**筛选器货架**可以添加交互式筛选：

1. **维度筛选**：拖入「渠道」→ 勾选要保留的值
2. **度量筛选**：拖入「GMV」→ 设置范围（如 GMV > 10000）
3. **日期筛选**：拖入「日期」→ 选择相对日期（如"最近 30 天"）

将筛选器**显示为交互控件**：右键筛选器 → "显示筛选器"。这样查看报表的人可以自己切换筛选条件。

### 参数

参数是用户可以动态输入的变量，比筛选器更灵活：

1. 右键数据面板 → "创建参数"
2. 设定名称、数据类型、允许值
3. 在计算字段中引用参数
4. 右键参数 → "显示参数控件"

**示例**：创建一个"目标值"参数，让用户自定义 GMV 目标线，并在图表中显示参考线。

## 构建仪表板（Dashboard）

仪表板是把多个工作表组合到一个页面上的交互式面板，是数据分析师交付给业务方的核心产出。

### 创建仪表板

1. 点击底部的"新建仪表板"图标
2. 设置仪表板大小（推荐固定大小：1200 × 800 或自动）
3. 从左侧将工作表拖入仪表板
4. 调整布局和大小

### 添加交互

仪表板的核心价值是**交互性**：

- **筛选器动作**：点击某个图表中的元素（如"微信"渠道），其他图表自动筛选到微信的数据
- **高亮动作**：悬停某个元素时，其他图表中对应数据高亮
- **URL 动作**：点击某个数据点跳转到详情页面

设置方式：仪表板菜单 → "动作" → 添加动作。

最常用的是**筛选器动作**，设置步骤：

1. 仪表板 → 动作 → 添加动作 → 筛选器
2. 源工作表：你要点击的图表
3. 目标工作表：你要被筛选的图表（选"全部"则影响所有图表）
4. 触发方式：选择"选择"（点击触发）

### 仪表板布局原则

| 原则 | 说明 |
|------|------|
| **从上到下，从左到右** | 符合阅读习惯，最重要的图表放左上角 |
| **大数字先行** | 顶部放 KPI 指标卡（GMV、订单量等大数字） |
| **趋势图放上半部分** | 折线图占据宽幅区域 |
| **对比图放下半部分** | 柱状图、排名表等放在下方 |
| **留白** | 不要塞太满，留出呼吸空间 |

## 实战：电商运营日报仪表板

用以下步骤完成一个完整的电商运营仪表板：

### 工作表清单

| 工作表 | 图表类型 | 展示内容 |
|-------|---------|---------|
| KPI 指标卡 | 文本/大数字 | 总 GMV、总订单、客单价、环比变化 |
| GMV 日趋势 | 折线图 | 近 30 天 GMV 走势 |
| 渠道对比 | 条形图 | 各渠道 GMV 排名 |
| 品类分布 | 饼图/树状图 | 品类 GMV 占比 |
| TOP 10 商品 | 表格 | 销量前 10 的商品 |
| 城市热力图 | 地图 | 各城市销售额分布 |

### KPI 指标卡做法

1. 新建工作表
2. 将「GMV」拖到**文本**标记卡
3. 右键 GMV → 格式 → 设置数字格式为"¥#,##0"
4. 调大字号（如 36pt），加粗
5. 添加环比计算字段，用不同颜色标识正负

### 发布与分享

完成仪表板后：

1. 文件 → 保存到 Tableau Public
2. 获得公开链接，可以嵌入网页或分享给同事
3. Tableau Public 主页就是你的数据分析作品集
